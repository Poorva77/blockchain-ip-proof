# Blockchain-Based IP Proof System

A demo project that creates proof of ownership for digital files using
SHA-256 hashing and blockchain timestamps.

## Features
- File upload
- Hash generation (SHA-256)
- Blockchain storage
- Duplicate detection

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js
- Blockchain: Ethereum (Hardhat)
- Smart Contract: Solidity
